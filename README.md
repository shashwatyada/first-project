# 📊 E-Commerce Sales Analysis using Excel

## 📌 Overview
This project analyzes e-commerce sales data using Advanced Excel tools like Pivot Tables, Charts, and Dashboards.

## 🎯 Objectives
- Analyze product-wise revenue
- Understand customer behavior
- Evaluate marketing channel performance
- Identify trends over time

## 🛠 Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Data Visualization

## 📊 Key Insights
- Highest Revenue Product: Chair & Printer
- Best Marketing Channel: Instagram
- Most Used Payment Method: Online
- High Return Rate in Laptops
- Peak Sales Month: June
